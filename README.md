FLE@M | Function of Labor-saving Equipment at Marketing
===============================================================================

学生時代に作成した、フリーマーケット運営支援用ソフトウェアです。

当時アルバイトをしていた某おもちゃ屋さんのレジシステムを参考に、
実際の運営におけるドメインのみにフォーカスして作成しました。

役に立つかはわかりませんが、掘り起こされたので共有します。
